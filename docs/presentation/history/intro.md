## Дневники

### Назначение дневников
 
Дневники предназначены для мониторинга состояния каких-либо показателей, называемых `переменные`  (variables).  
График может быть в виде:  
`0` - линейный  
`1` - линия прогресса  
`2` - круг прогресса  
`3` - число
