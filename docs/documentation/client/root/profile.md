# Профиль

> Исходный код: `ui/root/profile/profile.js`  

Профиль пользователя хранит личную информацию, такую как: 

- [x] ФИО  
- [ ] Место проживания  
- [x] Документы  
- [x] Анализы  
- [x] Медикаменты  

В верхнем углу окна есть кнопка перехода в настройки профиля, где можно изменить личную информацию.
