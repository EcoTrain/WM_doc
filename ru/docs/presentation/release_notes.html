Проверка наличия связи

При осутствии связи:

  • Не отправляются сообщения в чате
  • Все данные кэшируются на устройстве
  • Пользователь уведомляется в профиле

Оффлайн режим

  • Отстлеживание подключения к сети
  • Уведомление пользователя об отсутствии подключения к сети
  • Отслеживание ошибок сервера
  • Отслеживание отсутствие связи с сервером
  • Уведомление пользователя об ошибке передачи данных
  • Сохранение данных в очередь синхронизации
  • Синхронизация при появлении связи или перезагрузке приложения
  • Проверка на актуальность данных

Чат

  • Запрет отправки сообщений при отсутствии связи
  • Подсчёт непрочитанных сообщений
  • Синхронизация прочитанных сообщений при выборе чата
  • Автосинхронизация для текущего чата (если какой-либо чат открыт)
  • Уведомление пользователя об отсутствии связи с сервером (realtime)
